Instructions

To build your wishlist you need to click the '➕ ADD' link next to the games that you want to add This will create a private message for /u/nintendeals. If you send that message, the bot will add the game to your wishlist (for the selected country) and reply to you with your current wishlist. Later the bot, will notify you the next time that game goes on sale (via reddit private messages). Also, you can click on the '➕ ALL' link to add a game with all the available countries, not just the selected one. (Feel free to remove any country that you dont want).

In the messages that the bot sends, you'll see a ➖ icon next to each of your wishlisted games. If you click it, it will generate a private message for the bot to remove the selected game off your wishlist.

FAQ

Why another website for the game list?

Reddit limits the ammount of characters per post to 40k and the comments to 10k, so to fit every game with their ➕ link i would need something close to 3 posts or 13 comments which is a lot.

Would there be a limit for the ammount of games in the wishlist?

Yes, there is a limit of 50 games per user.

Why this have to be done with PMs?

This way you don't need to provider any information about your reddit account, everything is achieved with PMs between you and the bot. This website will only provide the format for the PMs but everything else is done securely inside the reddit.com site.
